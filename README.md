# tiendamatetp

Sitio estático listo para publicar en Netlify, GitHub Pages o Vercel.

## Incluye
- Catálogo con filtros y detalle de productos
- Constructor de combos que arma el pedido para WhatsApp
- Simulador de grabados personalizados
- Modo Campo / Noche
- Ambiente sonoro opcional generado en el navegador
- Animaciones, portada cinematográfica y efecto de profundidad
- WhatsApp, Instagram y Google Maps

Los precios actuales son provisorios y pueden reemplazarse más adelante desde `script.js`.
